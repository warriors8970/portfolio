CritiqueSuite

CritiqueSuite is an advanced review generation application that provides concise insights from just a single keyword. Originally developed for movie reviews, it now supports broader applications, including product reviews. Users can input a keyword—such as a movie title or product name—and receive a summarized, well-rounded review accompanied by a relevant image.

Technologies Utilized

  • Python: Primary programming language for development.
  • Transformers Library: Powers the advanced summarization models.
  • NLTK: For natural language processing, including tasks like sentence tokenization.
  • Google Custom Search API: Retrieves relevant images to complement the generated content.
  • Flask: Facilitates a streamlined web application environment.

Developing CritiqueSuite presented several challenges, especially in generating reviews that are both informative and engaging. By leveraging cutting-edge NLP techniques, I refined the app to meet high standards for quality and relevance. CritiqueSuite serves as a reliable tool for anyone seeking immediate, actionable insights—ideal for movie enthusiasts, shoppers, or any decision-makers.

Gesture-Controlled IoT: Enhancing Hands-Free Smart Home Interaction

This innovative project enables users to control household devices—such as TVs and air conditioners—through gesture detection, effectively transforming how we interact with smart home technology. Built on a Raspberry Pi equipped with a camera, this system captures real-time gestures and translates them into infrared (IR) commands for compatible devices.

Technologies Utilized

  • Raspberry Pi: The central processing unit, responsible for capturing gestures and transmitting commands.
  • OpenCV: Used for real-time video processing and gesture recognition.
  • MediaPipe: Facilitates efficient gesture detection through advanced machine learning models.
  • IR Transmitter: Sends out the configured IR codes to control devices remotely.

Implementation Details

The system is designed to recognize gestures corresponding to commands from an LG remote control. When a user performs a specific gesture, the camera captures this motion, and the model processes the gesture in real-time. Upon recognition, the Raspberry Pi transmits the appropriate IR code through its transmitter, allowing seamless control of the designated device.
